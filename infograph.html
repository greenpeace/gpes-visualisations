Infographics

With or without data, there are issues that can be better explained with an infographic made by a designer. If this infographic is going to be used in our site there’s some recommendations you must follow.

Source: Greenpeace • License: Creative Commons

Our static infographics are:

Editable with the best free open source software that exists for all operating systems (Inkscape) and with the most popular professional graphic design software (Adobe Illustrator). This compatibility allows people who’m we share the infographics to be able to edit them or reuse any component. SVG is a standard format, so it’s likely that it will open by other graphic editing software.

Vectorial, so you can easily adapt them to print at the highest quality. The same file that we publish on the web can be easily converted into a PDF and send to a printing company or printed in your own high quality inkjet or laser printer.

Responsive: the infographic adjusts itself to the screen size of your computer, mobile or tablet. Usually we make between 3 and 11 versions of the same infographic each one is optimised to a specific display size.

Accesible. Our infographics include the text and will begin to include image descriptions in the infographic. Part of the infographic can be understood by people with disabilities. Besides vectorial formats can be zoomed by users that need it.

Include reference to the typographic fonts and use our fonts that are open source. Although it’s not recommended, our infographics can use other fonts as long as we respect the font license.
