Unless it’s justified by other reasons, the typographic fonts we use in our graphics are the same as the official fonts for the site. We want to maintain visual consistency and reduce page loading times. All fonts are open source, so they can be downloaded and and used in any site, without purchasing licenses for each user.

Normally we will use the body text fonts, in this case "Work Sans",sans-serif for the main site and Merriweather,Georgia,serif for the magazine.

Transparency

Normally the transparency value should be 1 (fully opaque), but you can use the value 0.85 to see the grid or other elements in the back.
